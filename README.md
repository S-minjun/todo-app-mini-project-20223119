# Todo App - 풀스택 미니 프로젝트

## 📋 프로젝트 개요
React + Vite 프론트엔드와 Express + MongoDB 백엔드를 연결한 풀스택 Todo 리스트 애플리케이션입니다.

## 🚀 기능
- ✅ Todo 항목 추가
- 📝 Todo 목록 보기
- 🔄 Todo 완료 체크 (체크박스)
- 🗑️ Todo 삭제

## 🛠️ 기술 스택
- **Frontend**: React 19 + Vite + CSS + Axios
- **Backend**: Node.js + Express + MongoDB Atlas
- **배포**: Vercel

## 🔧 로컬 실행 방법

### 백엔드 실행
```bash
cd backend
npm install
npm run dev
```

### 프론트엔드 실행
```bash
cd frontend-react
npm install
npm run dev
```

브라우저에서 http://localhost:5173 접속
